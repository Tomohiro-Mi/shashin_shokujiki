# Shashoku Simulator (写植Sim)

ブラウザ上で動作する写真植字機（写植機）シミュレータです。
アナログな写植特有の「ボケ」「滲み」などの質感を再現し、物理的なダイヤル操作に近いUIで文字を打つ体験を提供します。

## 特徴 (Features)

*   **ローカルフォント利用 (Local Font Access)**:
    *   PCにインストールされているフォントをそのまま使用できます。
    *   Chrome/EdgeなどのChromium系ブラウザ推奨。
    *   フォントアクセスが制限される場合、`.ttf/.otf`ファイルを直接アップロードして使用するフォールバック機能もあります。
*   **アナログ質感の再現 (Analog Simulation)**:
    *   写真植字特有の、現像時の「角の丸み」や「インクの滲み」をCanvasフィルタでリアルタイムにシミュレートしています。
*   **こだわりのUI (Authentic UI)**:
    *   **メインレンズ (Main Lens)**: Q数（級数）単位での文字サイズ変更 (12Q - 100Q)。
    *   **採字版 (Main Plate)**: ひらがな、カタカナ、漢字、記号をタブで切り替え可能。
    *   **カスタム文字追加**: グリッドにない文字（珍しい漢字や絵文字など）も自由に追加可能。
    *   **送り操作**: 文字送り（Okuri）や行送り（Feed）をH（歯）単位で調整可能。
*   **出力機能 (Export)**:
    *   作成した印画紙を **PNG画像 (現像)** または **PDF (校正刷り)** として保存できます。

## 使い方 (Usage)

1.  `index.html` をブラウザで開きます。
    *   ※ ローカルフォントAPIの制限により、ローカルサーバー経由（例: `npx serve .` や VS CodeのLive Server）での起動を推奨しますが、直接ファイルを開いても動作するように調整済みです。
2.  **Load System Fonts** ボタンを押し、システムフォントへのアクセスを許可します。
3.  使用したいフォントを選択します（ダブルクリックまたはSelectボタン）。
4.  **メインプレート**から文字をクリック、または **カスタム入力欄** に文字を入れて追加し、赤い **SHUTTER** ボタンを押して印字します。
5.  完成したら **Export PNG** または **Export PDF** で保存します。

## 動作環境 (Requirements)

*   **推奨ブラウザ**: Google Chrome, Microsoft Edge (PC版)
    *   *Local Font Access API* をサポートしているブラウザでフル機能が使えます。
